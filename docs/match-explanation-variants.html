매칭 결과 — "4개가 뭔데?" 설명 추가 방식 비교

유저가 "4개 비교"가 구체적으로 무엇인지 알 수 있게 설명을 넣는 3가지 방식. 같은 트렌드 A(매트 쿠션, 4/4 강 매칭)로 그려서 비교합니다.

α

ⓘ 호버 툴팁

각 행 우측 ⓘ 아이콘에 마우스 올리면 설명 표시. 시각적 노이즈 최소.

매칭 결과 G5
4개 비교 중 4개 통과
브랜드 적합성 브랜드 톤·키워드·제형이 트렌드의 흐름·키워드와 자연스럽게 맞는지 2가지로 확인합니다. 2/2
타겟 적합성 우리 타겟 고객이 이 트렌드의 주 소비층·라이프스타일과 일치하는지 2가지로 확인합니다. 2/2
β ★추천

행 아래 한 줄 부연

각 행 바로 아래 작은 회색 글씨로 무엇을 봤는지 항상 노출. 인쇄·스크린샷에도 보임.

브랜드 적합성 2/2
톤·키워드가 트렌드와 맞는지
타겟 적합성 2/2
우리 고객층·라이프스타일이 트렌드와 맞는지
γ

접히는 자세히 보기

평소엔 접혀 있고 "어떻게 평가했나요?" 클릭 시 4가지 비교 전체 펼침. 깊이 있는 정보, 평소 화면은 깨끗.

브랜드 적합성 (2개)
  • 브랜드 톤앤매너 ↔ 트렌드 흐름
  • 브랜드 키워드·제형 ↔ 트렌드 키워드
타겟 적합성 (2개)
  • 우리 타겟 ↔ 트렌드 주 소비층
  • 우리 타겟 ↔ 트렌드 라이프스타일·동기
