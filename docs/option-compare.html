옵션 비교 — G5 (d) 매칭 강도 vs (e) 채널 현황

audience % 데이터의 신뢰성 문제로 (a)·(b)·(c)는 제외. 대안 두 가지를 실제 그려서 비교합니다.

왜 (a)·(c)도 빠졌나: 매칭가의 2-A reason에 "구체 수치 비공개"라고 적힌 이유는 audience 데이터(특히 성별)가 네이버 데이터랩 등에서 정확히 안 나와서입니다. 트렌드 분석가가 audience_distribution 필드에 값을 채우긴 하지만 추정치 성격이라 정확한 %를 노출하면 오해 소지가 있어요. 그래서 (c) 코호트 %, (a) 로컬 계산 % 모두 정직한 표현이 아닙니다. G1 스택바도 hover tooltip의 % 라벨은 빼는 게 맞고, 시각 비율로만 보여주는 게 좋습니다.
